Mac Filezilla 报错的解决办法

2018-06-07

Mac Filezilla：无法写入“../.config/filezilla/filezilla.xml”

出现原因：.config 目录权限不足

解决办法：

  1. 用户根目录执行：

    sudo chmod -R 777 .config

  2. 进入 .config 后创建 filezilla 目录：

    mkdir filezilla

  3. 赋予 filezilla 目录 777 权限：

    sudo chmod -R 777 .config/filezilla
